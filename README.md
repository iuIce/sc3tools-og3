[CommitteeOfZero/sc3tools](https://github.com/CommitteeOfZero/sc3tools) for Oregairu Kan, with a reworked sg0 profile-based charset.<br>
The original charset ends at line 7552, any new characters must also be added to the bitmap fonts.

The `<vtext>` tag will be replaced with [linebreak] when replacing text in the .msb files.
